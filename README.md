# Autonomous Weeds Detection 🌱

This project uses YOLOv8 to detect weeds in agricultural fields.

## Features
- Object detection using YOLOv8
- Dataset trained on weed images
- Real-time prediction support

## Tech Stack
- Python
- YOLOv8
- Google Colab

## How to Run
1. Open notebook in Colab
2. Install dependencies
3. Run all cells
